Common Filters

Grayscale


In order to grayscale an image, what you must do is get the average of the rgb values across the pixel, and then apply the average to r, g, and b

Sepia Tone


In order to make an image appear sepia toned, we use an algorithm that uses math to alter our rgb values to have that brownish hue.

Sobel's Edge Detection


The Sobel operator is an edge detection algorithm. Using it, we can adjust the brightness of each pixel according to how much it seems to be an edge.
